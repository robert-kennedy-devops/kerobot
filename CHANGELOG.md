# Changelog

Todas as mudanças relevantes deste projeto serão documentadas neste arquivo.

## [0.1.0] - 2026-03-14
### Added
- Cliente MTProto com automação de botões inline.
- Engine baseada em eventos com fila de ações e anti-flood.
- Workers concorrentes (caça, combate, cura, poções, masmorra).
- PostgreSQL com migrations e configuração por usuário.
- Bot de configuração com login por QR.
- Modo captura e regras aprendidas.
- Docker e Docker Compose.
- Métricas em JSON.
